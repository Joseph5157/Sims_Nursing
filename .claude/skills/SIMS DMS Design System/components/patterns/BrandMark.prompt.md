**BrandMark** — the app's identity mark: 🎓 on the blue→indigo gradient tile, optionally with the "SIMS DMS" wordmark.

```jsx
<BrandMark />
<BrandMark showWordmark />
<BrandMark size="lg" glow />   {/* login hero */}
```

There is no standalone logo file — always render the mark from this component (or the gradient tokens) so it stays consistent.
