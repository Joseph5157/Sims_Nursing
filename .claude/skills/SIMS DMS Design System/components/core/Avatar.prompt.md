**Avatar** — initials badge from a person's name. Two skins: gradient fill (default, on light surfaces) and a ringed blue treatment for the dark sidebar.

```jsx
<Avatar name="Priya Sharma" />
<Avatar name="Joseph K" size="lg" />
<Avatar name="Admin User" onDark />
```

Sizes: `sm` (30px), `default` (38px), `lg` (48px). Shows the first two initials, uppercased.
